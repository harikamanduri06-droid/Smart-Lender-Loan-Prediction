# Smart Lender – Loan Eligibility Prediction

## 📌 Project Overview
This project predicts whether an applicant is eligible for a loan using Machine Learning algorithms. It was developed as part of the SmartBridge program.
## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Git
- GitHub
## 📂 Repository Contents
- `Loan Prediction using ML.ipynb` – Jupyter Notebook containing data preprocessing, analysis, model building, and results.
- `ERD.png` – Entity Relationship Diagram for the project.
